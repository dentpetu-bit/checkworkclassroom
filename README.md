# QR Score Studio by Kru Dew — v6 Score Import

เวอร์ชันนี้เพิ่มระบบนำเข้าคะแนนจากภายนอก พร้อม Template คะแนนรายห้อง

## เพิ่มใหม่ใน v6
- ดาวน์โหลด Template คะแนนจากหน้ารายงาน
- นำเข้าคะแนนจาก Excel / CSV
- วางข้อมูลจาก Google Sheets หรือ Excel ได้
- รองรับตารางกว้างตาม Template: รหัสนักเรียน + คอลัมน์ชิ้นงานหลายงาน
- รองรับแบบรายการ: รหัสนักเรียน / ชิ้นงาน / คะแนน
- นำเข้าแล้วอัปเดตคะแนนเดิมใน Supabase อัตโนมัติ
- แยกชิ้นงานตามห้องเหมือน v5

## วิธีใช้
1. อัปโหลดไฟล์ทั้งหมดขึ้น GitHub ทับของเดิม
2. เปิดเว็บแล้วกด `Ctrl + Shift + R`
3. ไปที่หน้า `รายงาน/นำเข้าคะแนน`
4. เลือกห้อง
5. กด `ดาวน์โหลด Template คะแนน`
6. กรอกคะแนนในคอลัมน์ชิ้นงาน
7. อัปโหลดไฟล์กลับเข้าระบบ
8. กด `ตรวจสอบคะแนน`
9. กด `นำเข้า / อัปเดตคะแนน`

## หมายเหตุ
- ช่องว่างใน Template จะไม่ถูกนำเข้าและไม่แก้คะแนนเดิม
- ถ้าต้องการบันทึก 0 คะแนน ให้พิมพ์เลข `0` ลงในช่องนั้น
- ระบบจะจับคู่จาก `รหัสนักเรียน` และชื่อคอลัมน์ชิ้นงานของห้องที่เลือก
