# QR Score Studio v4 by Kru Dew

เวอร์ชันนี้เพิ่มตามคำขอ:

- จัดการชิ้นงาน: เพิ่ม / แก้ไข / ลบ / เลื่อนลำดับขึ้นลง
- รายงานภาพรวม: แก้ไขคะแนนในตารางได้ทันที กด Enter หรือคลิกออกจากช่อง ระบบบันทึกลง Supabase อัตโนมัติ
- ช่องคะแนนที่ยังไม่ได้กรอกจะแสดงค่า 0 และพื้นหลังสีแดงอ่อน
- หลังสแกน QR นักเรียน จะแสดงชื่อนักเรียนตัวใหญ่กลางจอ
- หลังสแกน QR คะแนน จะแสดงคะแนนตัวใหญ่กลางจอ
- ยังรองรับจัดการนักเรียนและนำเข้า Excel/CSV ผ่านหน้าเว็บเหมือนเดิม

## วิธีอัปเดต

1. อัปโหลดไฟล์ทั้งหมดขึ้น GitHub ทับของเดิม
2. เข้า Supabase > SQL Editor
3. รันไฟล์ `schema.sql` อีกครั้ง เพื่อเพิ่ม policy สำหรับแก้ไข/ลบชิ้นงาน
4. เปิดเว็บแล้วกด `Ctrl + Shift + R`

## หมายเหตุ

ถ้าลบชิ้นงาน คะแนนของชิ้นงานนั้นจะถูกลบตามไปด้วย เพราะ `scores.assignment_id` ตั้งค่า `on delete cascade`
