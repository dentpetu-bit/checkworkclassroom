# QR Score Studio by Kru Dew — v11 Barcode Scanner Mode

เพิ่มโหมดรองรับเครื่องยิงบาร์โค้ด/QR แบบ Keyboard Wedge

## วิธีใช้
1. อัปโหลดไฟล์ทั้งหมดขึ้น GitHub ทับของเดิม
2. เปิดเว็บด้วย `?v=11` หรือกด Ctrl + Shift + R
3. ไปหน้า `สแกนคะแนน`
4. เลือกห้องและชิ้นงาน
5. ใช้เครื่องยิงรหัสนักเรียน แล้วยิงคะแนนต่อได้เลย

ระบบรองรับทั้งกล้องเดิมและเครื่องยิงบาร์โค้ด/QR โดยส่วนอื่นยังเหมือนเดิม
