# QR Score Studio by Kru Dew — v9 Export Fix

เวอร์ชันนี้แก้ปัญหาหน้ารายงาน:

- ส่งออกภาพ PNG ได้เต็มตาราง ไม่ตัดตามกรอบเลื่อน
- ส่งออก Excel แล้วมีคะแนนทุกช่อง ไม่เหลือแค่คะแนนรวม
- ช่อง 0 คะแนนยังเป็นสีแดง
- คะแนนเต็มยังเป็นสีเขียว
- เก็บฟีเจอร์ v8 เดิมทั้งหมด เช่น แยกชิ้นงานตามห้อง เพิ่ม/ลบ/แก้ไข/เลื่อนลำดับชิ้นงาน นำเข้านักเรียน และนำเข้าคะแนน

## วิธีใช้
1. อัปโหลดไฟล์ทั้งหมดขึ้น GitHub ทับของเดิม
2. เปิดเว็บด้วย `?v=9` หรือกด Ctrl + Shift + R
3. ไปหน้า รายงาน/นำเข้าคะแนน
4. โหลดรายงาน แล้วกด Excel หรือ ภาพ PNG
