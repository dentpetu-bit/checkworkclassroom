# QR Score Studio by Kru Dew — v10 Duplicate Score Warning

เพิ่มระบบแจ้งเตือนเมื่อสแกน/กรอกคะแนนซ้ำ

## เพิ่มใหม่
- ถ้านักเรียนมีคะแนนเดิมในชิ้นงานนั้นแล้ว และคะแนนเดิมไม่ใช่ 0 ระบบจะขึ้นแจ้งเตือน
- ถ้าครูกด Cancel ระบบจะไม่บันทึกทับ
- ถ้ากด OK ระบบจะบันทึกทับคะแนนเดิม
- หน้าแก้คะแนนรายงาน ถ้าแก้จากคะแนนเดิมที่ไม่ใช่ 0 จะถามยืนยันก่อน

## วิธีใช้
1. อัปโหลดไฟล์ทั้งหมดขึ้น GitHub ทับของเดิม
2. เปิดเว็บด้วย `?v=10` หรือกด Ctrl + Shift + R
