# QR Score Studio by Kru Dew

เว็บแอปบันทึกคะแนนด้วย QR Code สำหรับรันบน GitHub Pages และบันทึกข้อมูลลง Supabase

## ความสามารถ
- เลือกห้องเรียนและชิ้นงาน
- สแกน QR รหัสนักเรียน แล้วระบบดึงชื่อนักเรียนขึ้นทันที
- สแกน QR คะแนนตัวเลข แล้วบันทึกคะแนนลง Supabase อัตโนมัติ
- กล้องกลับไปรอการสแกนนักเรียนคนต่อไปเอง
- กรอกคะแนนเองแบบ Manual ได้
- เพิ่มชิ้นงานได้
- รายงานภาพรวมรายห้อง เรียงชิ้นงาน งาน 1, งาน 2, งาน 3...
- ส่งออก Excel และ PNG ได้

## วิธีติดตั้ง
1. สร้าง Supabase Project
2. เปิด SQL Editor แล้วรันไฟล์ `schema.sql`
3. เพิ่มรายชื่อนักเรียนในตาราง `students`
4. คัดลอก `config.example.js` เป็น `config.js`
5. ใส่ `SUPABASE_URL` และ `SUPABASE_ANON_KEY`
6. อัปโหลดไฟล์ทั้งหมดขึ้น GitHub repository
7. เปิด Settings > Pages > Deploy from branch > เลือก branch `main` และ `/root`

## รูปแบบ QR Code
- QR นักเรียน: ใส่รหัสนักเรียน เช่น `40201` หรือ `student:40201`
- QR คะแนน: ใส่ตัวเลข เช่น `10` หรือ `score:9.5`

## หมายเหตุความปลอดภัย
เวอร์ชันนี้ออกแบบให้ใช้ง่ายในห้องเรียน โดยเปิด policy ให้ anon key อ่าน/เขียนได้ ถ้าต้องการใช้จริงระยะยาว แนะนำเพิ่มระบบ Login ครูด้วย Supabase Auth แล้วปรับ RLS policy เป็น authenticated เท่านั้น
