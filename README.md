# QR Score Studio v15 — คะแนนจริงและเกรด

เพิ่มเมนู “คะแนนจริง”

- เลือกห้อง
- กำหนดคะแนนเต็ม 4 ช่อง: ก่อนกลางภาค / กลางภาค / หลังกลางภาค / ปลายภาค
- เลือกชิ้นงานดิบหลายงานในแต่ละช่อง
- ระบบคำนวณคะแนนจริง = คะแนนดิบที่ได้รวม / คะแนนเต็มของงานรวม × คะแนนเต็มช่อง
- รวมคะแนน 4 ช่อง
- ประเมินเกรด 4.0 ถึง 0 ตามช่วงคะแนนที่กำหนด
- ส่งออก Excel ได้
