# QR Score Studio v5 — แยกชิ้นงานตามห้อง

เวอร์ชันนี้เพิ่มระบบแยกชิ้นงานตามห้องเรียน เช่น งานของ 4/2 จะไม่ไปปนกับ 4/8 หรือ 6/6

## วิธีอัปเดต
1. อัปโหลดไฟล์ทั้งหมดขึ้น GitHub ทับของเดิม
2. เข้า Supabase > SQL Editor
3. รันไฟล์ `schema.sql` 1 ครั้ง เพื่อเพิ่มคอลัมน์ `room` ในตาราง `assignments`
4. กลับไปหน้าเว็บแล้วกด `Ctrl + Shift + R`

## สิ่งที่เพิ่มใน v5
- หน้าเพิ่มชิ้นงานมีช่องเลือกห้องของชิ้นงาน
- หน้าแสกนคะแนนจะแสดงเฉพาะชิ้นงานของห้องที่เลือก
- หน้ารายงานจะแสดงเฉพาะชิ้นงานของห้องรายงานนั้น
- งานของแต่ละห้องลบ/แก้ไข/เลื่อนลำดับได้แยกกัน

## หมายเหตุ
ชิ้นงานเดิมที่เคยสร้างไว้ก่อนอัปเดต จะถูกย้ายไปอยู่ห้อง `4/2` อัตโนมัติจาก SQL migration
