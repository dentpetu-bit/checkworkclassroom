# QR Score Studio by Kru Dew — v8 Sort Order Fix

เวอร์ชันนี้แก้ปัญหาเพิ่มชิ้นงานแล้วขึ้น `sort_order violates not-null constraint` และแก้เลขลำดับชิ้นงานกระโดด

## ต้องทำ
1. อัปโหลดไฟล์ทั้งหมดขึ้น GitHub ทับของเดิม
2. เปิด Supabase SQL Editor แล้วรัน `schema.sql`
3. เปิดเว็บด้วย `?v=8` หรือกด Ctrl + Shift + R

## สิ่งที่แก้
- `sort_order` ไม่ใช้ identity แล้ว จึงแก้ลำดับได้
- เพิ่มชิ้นงานใหม่ใช้เลขลำดับล่าสุดของห้องนั้น + 1
- จัดลำดับงานเดิมใหม่แยกตามห้อง เริ่ม 1,2,3...
- ปุ่ม ↑ ↓ ยังใช้งานได้
